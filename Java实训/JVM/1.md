# 虚拟机

可以运行完整操作系统的软件平台，专门为 执行单个计算机程序而设计，执行的指令是我们称的java字节码指令

jvm java virtual machine

执行字节码 扩语言的平台

二进制的运行环境

## 特点

1、一次编译，导出运行

2、自动内存管理

3、自动垃圾回收功能
